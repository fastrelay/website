Fast Relay

Making it easier to run your own Tor Relay.

Features Download

  • Fasto? Relayo?

    FastRelay is a quick and easy way to setup a Tor Relay. Along with multiple plugin support to help you with operating your relay.

  • Multi-Branch Install base

    Whether it's the Experimental, Nightly, or just plain Stable build. You have the option to pick which one you'd like to install for your needs!

How it works

Integration

With a web server like Nginx, and a resource monitor like Nyx. You can now display simple web pages with Nginx stating the practices/uses of the Tor Relay, and Monitor your Relays health at the same time by using Nyx!

Blocking

With advance blocking with strict ExitPolicies you can now have more options to (BLOCK, DROP, REJECT) certain IP's or complete IP Subnets. As well as blocking or setting strict allow Port rules for your Relay.
